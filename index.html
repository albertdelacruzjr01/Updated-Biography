Chillax

Hello! This is by far one of the most challenging websites I've created. I've created a simple social media website. It's a bit similar to some functionalities of Facebook with my personal touch.

I made the name as "Chillax" since I want the users to be able to simply chill and relax while browsing my website. I inserted some icons for computers that are touch screen. I also made sure that it will be a very user-friendly website to everyone.

In this website, you will be able to register with verification email attached to it, reset password, post something with image(optional), update your profile, insert your own profile picture, comment, like, delete your own post/comment. I also made sure that the only user who can delete post is the same user who created it. That goes the same with comments. Likes accumulate as count. You will be able to add friends, unfriend and will be able to view the list of your friends in the "Friends" section. Unfortunately, the messaging section isn't working yet since we have a deadline but will be finishing it soon.

Lastly, the programming language I used is Laravel 5.7 and by far, my favorite framework. It has multiple one-to-one, one-to-many and many-to-many relationships attached to it. It was very challenging by with patience, I was able to build it.

Hope you enjoy! :)
